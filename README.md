Bubble
======

![Bubble](Bubble.png)

A microframework for Pharo.

A bubble is an application.

At this point, only the command line driver works.

## Getting this to run

To float (run) a Bubble, do:

    ./bubble --app file.bubble

To launch the IDE, do:

    ./bubbleui &

## Tools

See command line options:

    ./bubble --help
